## üöÄ Quick Start Path

### Estimated Time: 15-30 minutes

**Step 1: Read (5 min)**
```markdown
Start: README.md or COMPREHENSIVE_DOCUMENTATION.md (Quick Start section)
```

**Step 2: Install (10 min)**
```bash
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

**Step 3: Configure (5 min)**
```bash
cp .env.example .env
# Edit .env with your API keys
```

**Step 4: Launch (5 min)**
```bash
# Windows
run_dashboard.bat

# Linux/Mac
python dashboard/main.py
```

**Step 5: Process Data (5+ min)**
- Add symbols
- Click Start
- Watch progress
